# تطبيق توليد بطاقات الجرد بصيغة PDF

تطبيق ويب مبني باستخدام `Streamlit` لتحويل ملفات الجرد بصيغة `Excel` إلى بطاقات جرد بصيغة `PDF` جاهزة للطباعة، مع دعم اللغة العربية، اتجاه الكتابة من اليمين إلى اليسار، وإمكانية استعمال ترويسة رسمية ثابتة أو رفع ترويسة خاصة بالمؤسسة.

هذا المشروع صُمم ليساعد المؤسسات التعليمية والإدارية على إعداد بطاقات توطين المجرود بطريقة أسرع، أوضح، وأكثر احترافية من العمل اليدوي داخل ملفات `Excel`.

## ما الذي يقدمه هذا المشروع؟

يقوم التطبيق بقراءة ملف الجرد، ثم يحلل الأعمدة التي تمثل القاعات أو المختبرات أو الفضاءات المختلفة، وبعدها ينشئ بطاقة مستقلة لكل قاعة داخل ملف `PDF` منسق وجاهز للطباعة.

يسمح التطبيق أيضًا بما يلي:

- استعمال ترويسة رسمية ثابتة من داخل المشروع
- رفع ترويسة خاصة بالمستخدم
- إدخال معلومات المؤسسة داخل الترويسة
- تحميل قالب `Excel` جاهز للتعبئة
- تصدير جميع البطاقات داخل ملف `ZIP`

## كيف يعمل؟

فكرة العمل بسيطة:

1. يرفع المستخدم ملف الجرد بصيغة `Excel`
2. يحدد التطبيق سطر العناوين تلقائيًا
3. يكتشف أعمدة القاعات مثل `Salle` و`قاعة` و`مختبر`
4. يختار المستخدم عمود اسم التجهيز أو الأثاث
5. يولد التطبيق بطاقة `PDF` لكل قاعة
6. يتم تنزيل جميع البطاقات داخل ملف مضغوط واحد

## المزايا الرئيسية

- واجهة عربية مع دعم `RTL`
- إخراج نهائي بصيغة `PDF`
- دعم الترويسة الثابتة أو الترويسة المرفوعة
- دعم الخطوط العربية داخل المستند
- توفير قالب `Excel` جاهز
- مناسب للنشر على `Streamlit Community Cloud`

## بنية الإدخال

يجب أن يحتوي ملف الجرد عادة على:

- عمود خاص باسم التجهيز أو الأثاث
- أعمدة تمثل القاعات أو المختبرات
- أعداد التجهيزات داخل كل قاعة

أمثلة على أسماء الأعمدة التي يتعرف عليها التطبيق:

- `بيان التجهيز / الأثاث`
- `Salle 01`
- `Salle 02`
- `مختبر`
- `SVT`
- `PC`

## طريقة الاستخدام

1. شغل التطبيق
2. اختر طريقة العمل:
   `رفع ملف جرد جاهز`
   `تحميل قالب Excel ثم تعبئته`
3. إذا لم يكن لديك ملف جاهز، قم بتحميل القالب ثم عبئه
4. أدخل:
   `اسم المؤسسة`
   `المديرية`
   `الأكاديمية / الجهة`
   `تاريخ التحيين`
5. اختر وضع الترويسة:
   `استعمال الترويسة الثابتة مع إضافة المعلومات`
   `رفع ترويسة خاصة`
6. ارفع ملف الجرد
7. اختر العمود الذي يحتوي على اسم التجهيز
8. اضغط على زر توليد البطاقات
9. حمّل ملف `ZIP` الناتج

3. اختر المستودع
4. اجعل ملف التشغيل الرئيسي هو:

```text

```

5. انشر التطبيق
