# تطبيق توليد بطاقات الجرد PDF

تطبيق مبني باستخدام `Streamlit` يهدف إلى تحويل بيانات الجرد الموجودة في ملف `Excel` إلى بطاقات جرد بصيغة `PDF` جاهزة للطباعة، مع دعم اللغة العربية واتجاه الكتابة من اليمين إلى اليسار.

هذا التطبيق مناسب للمؤسسات التعليمية أو الإدارية التي تحتاج إلى إعداد بطاقات توطين المجرود لكل قاعة أو مرفق بشكل منظم ورسمي.

## فكرة المشروع

يقوم المستخدم برفع ملف الجرد بصيغة `Excel`، ثم يقرأ التطبيق البيانات ويستخرج القاعات أو المختبرات تلقائيًا، وبعد ذلك ينشئ بطاقة جرد مستقلة لكل قاعة داخل ملف `PDF`.

يمكن كذلك:

- رفع ترويسة رسمية أو شعار المؤسسة ليظهر أعلى الصفحة
- تحميل قالب `Excel` جاهز للاستعمال ثم تعبئته وإعادة رفعه
- تنزيل جميع البطاقات في ملف `ZIP` واحد

## المزايا

- واجهة عربية مع دعم `RTL`
- توليد ملفات `PDF` جاهزة للطباعة
- اكتشاف تلقائي لأعمدة القاعات مثل `Salle` و`قاعة` و`مختبر`
- دعم رفع ترويسة أو شعار رسمي للمؤسسة
- توفير قالب `Excel` جاهز لتعبئة بيانات الجرد
- تصدير جميع البطاقات داخل ملف مضغوط `ZIP`

## طريقة العمل

1. افتح التطبيق.
2. اختر إن كنت تريد:
   - رفع ملف جرد جاهز
   - أو تحميل قالب Excel ثم تعبئته
3. ارفع ملف الجرد بصيغة `xlsx` أو `xls`
4. أدخل اسم المؤسسة وتاريخ التحيين
5. ارفع الترويسة الرسمية أو شعار المؤسسة إذا رغبت بذلك
6. اختر العمود الذي يحتوي على اسم التجهيز أو الأثاث
7. اضغط على زر توليد البطاقات
8. قم بتحميل ملف `ZIP` الذي يحتوي على جميع بطاقات الجرد بصيغة `PDF`

## بنية ملف Excel

يُفضّل أن يحتوي الملف على:

- عمود خاص باسم التجهيز أو الأثاث
- أعمدة تمثل القاعات أو المختبرات
- القيم داخل الأعمدة تكون عبارة عن عدد التجهيزات داخل كل قاعة

مثال على أسماء الأعمدة المقبولة:

- `بيان التجهيز / الأثاث`
- `Salle 01`
- `Salle 02`
- `مختبر`
- `SVT`
- `PC`

بعد تثبيت المتطلبات، شغّل التطبيق بالأمر التالي:

```bash

```
